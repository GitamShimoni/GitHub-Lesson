Question 1: Factorial Calculation Using Recursion
Write a recursive function in C to calculate the factorial of a number. The function should take an integer n as input and return the factorial of n.

Example Input: n = 5
Example Output: Factorial of 5 is 120
Bonus: Write a loop-based implementation for comparison.

Question 2: Check if a String is a Palindrome
Write a recursive function to check if a given string is a palindrome. The function should take a string and its start and end indices as input. Return 1 if it is a palindrome and 0 otherwise.

Example Input: "madam"
Example Output: "madam is a palindrome"
Hint: Compare the first and last characters and recursively check the remaining substring.

Question 3: Sum of Digits Using Recursion
Write a recursive function to calculate the sum of the digits of a given positive integer.

Example Input: n = 1234
Example Output: Sum of digits = 10
Hint: Use the modulo operator % to extract the last digit and integer division / to remove it.

Question 4: Print Fibonacci Series Up to N Terms
Write a program to print the first n terms of the Fibonacci series using recursion and loops. The Fibonacci series is defined as:
F(n)=F(n−1)+F(n−2),for n≥2

Example Input: n = 7
Example Output: 0 1 1 2 3 5 8
Bonus: Optimize the solution by using a loop for generating the series and compare it with the recursive approach.
